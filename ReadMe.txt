In order to run the code you must have the files featureVector.mat, findFeat.m, Dip_Final_Project.m, and the image library Fruit Database that holds all the jpegs. In order to run the code, have all of these files in the current folder and run the Dip_Final_Project.m file. In order to change the images, go to line 6 in Dip_Final_Project.m and change the image that one would like to use.
